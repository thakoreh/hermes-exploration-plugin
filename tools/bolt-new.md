# Bolt (bolt.new)

Bolt.new is an AI coding environment from StackBlitz that runs entirely in your browser. Describe a feature or paste a screenshot, and Bolt generates a working full-stack app with auth, database, and deployment. Built on WebContainer (real Node.js in browser).

## Key Features

- **Full-stack in browser** — Node.js backend, React/Vue/Svelte frontend, all in-browser
- **AI chat-to-code** — describe features, get working code
- **One-click deploy** — straight to production URL
- **Real file system** — edit any file, install packages, run scripts
- **Preview alongside code** — split-pane editor + live preview
- **Database included** — SQLite, Postgres options via integrate
- **Auth scaffolding** — add authentication with one click

## vs Alternatives

- **vs Lovable**: More focused on full-stack, slightly better for backend
- **vs Replit**: Browser-native, no setup, faster for quick prototypes
- **vs Cursor**: More opinionated templates, faster for pure prototyping

## Use Cases

- MVPs and hackathon projects
- Rapid feature prototyping before full implementation
- Testing library/framework ideas
- Demo apps for client presentations
