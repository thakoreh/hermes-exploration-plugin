# Windsurf

Windsurf is Codeium's AI IDE featuring a unique "Cascade" architecture. The primary agent handles routine tasks; for complex multi-step work, it escalates to a more powerful agent with full context.

## Cascade Architecture

- **Primary Agent**: Handles 80% of tasks — autocomplete, quick edits, inline chat
- **Cascade Agent**: Takes over complex tasks, maintains full project state, can call tools recursively
- **Supercompletion**: Suggests entire blocks/flows, not just next line

## Key Features

- **Supercomplete** — predicts entire code blocks, not just next token
- **Project-aware chat** — understands your full repo structure
- **Tool use** — Cascade agent can run terminal commands, search files, execute tasks
- **Multi-file editing** — edit across many files in one Cascade session
- **Flow mode** — step-by-step AI guidance for complex changes
- **Context persistence** — AI remembers conversation history within a session
