# Cursor

Cursor is an AI-first code editor built on VS Code. It combines a powerful editor with AI that understands your entire codebase — not just the current file. The AI can search, read, and edit across thousands of files.

## Key Features

- **Tab autocomplete** — learns from your codebase, predicts next edits
- **AI Chat** — ask about any part of your codebase (Ctrl+L / Cmd+L)
- **Composer** — generate entire files/features via AI
- **Agent mode** — AI can edit multiple files, run commands, fix bugs
- **Codebase indexing** — AI reads and understands your full project
- **Privacy mode** — disable data collection, enterprise options
- **Vision support** — paste screenshots, AI understands UI designs

## Cursor Rules (custom instructions)

Set persistent AI behavior rules stored in `.cursorrules` file:
- Coding style and conventions
- Tech stack preferences
- Testing requirements
- Documentation standards

## Plans

- **Free**: 100 slow AI requests/mo
- **Pro**: $20/mo, unlimited fast AI
- **Business**: $30/mo, team features + SSO
- **Enterprise**: Custom pricing

## Keyboard Shortcuts

- `Ctrl/Cmd + L` — AI chat
- `Tab` — Accept AI suggestion
- `Ctrl/Cmd + K` — Edit inline
- `Ctrl/Cmd + Enter` — Generate code at cursor
